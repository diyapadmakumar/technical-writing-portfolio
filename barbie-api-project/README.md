# Barbie Store API Documentation
This project contains the complete documentation for the Barbie Store REST API. It follows industry standards like **RFC 7807** for error handling and the **Google Developer Style Guide** for technical clarity.

---

## Documentation Suite
Explore the different sections of the documentation below:

1. **[Quickstart Guide](./quickstart.md)** *Get up and running with your first API request in under 5 minutes.*

2. **[API Reference](./api-reference.md)** *Detailed technical specifications for every endpoint, parameter, and response.*

3. **[Style Guide](./style-guide.md)** *Audience persona, voice and tone benchmarks, and the technical rationale behind our terminology choices.*

3. **[Error Reference Guide](./error-reference-guide.md)** *A complete guide to HTTP status codes and resolution steps.*

## Tools Used
* **cURL:** For testing endpoint connectivity and verifying API responses via the terminal.
* **Markdown:** For structured, version-controlled documentation.
* **Git/GitHub:** For documentation-as-code workflow.
